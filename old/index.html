CoreWar

Language
C
Context
University second semester project - made alone
Date, Progression
may 2009
Description
Core War (or Core Wars) is a programming game in which two or more battle programs (called "warriors") compete for the control of the "Memory Array Redcode Simulator" virtual computer ("MARS"). These battle programs are written in an abstract assembly language called Redcode. The object of the game is to cause all processes of the opposing program(s) to terminate, leaving your program in sole possession of the machine. (source)
This CoreWar Project has 9 instructions. Each instruction is represented on 32 bits and there are 4096 locations in the virtual machine memory.
License
GPL
Links
Project source code on github
